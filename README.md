## 🚀 빠른 시작 (수강생)

1. **사전 준비** 탭에서 본인 OS에 맞춰 Claude Code 설치
2. **실습 파일** 탭에서 Excel 3개 다운로드 → `shareclass-실습/` 폴더에 저장
3. **실습 1~4** 탭을 따라가며 한 줄씩 프롬프트 입력
4. 막히면 보조강사에게 손, 또는 화면에 직접 "도와줘" 입력
5. 강의 종료 후 `HOMEWORK_심화실습.md` 에서 본인 직무에 가까운 과제 1~2개 선택해 자율 진행
