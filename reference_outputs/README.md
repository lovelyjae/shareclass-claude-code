## 시연 전 강사 체크리스트

- [ ] `files/` 3개 파일이 `shareclass-실습/` 폴더에 복사되었는지
- [ ] 본인 노트북에서 동일 프롬프트 → 동일 결과 나오는지 1회 시험
- [ ] Streamlit이 사전에 설치되어 있는지 (`pip install streamlit altair`)
- [ ] 결과물(.md)이 바탕화면이 아닌 `shareclass-실습/`에 생성되는지

## 데이터 특징

- **매출 데이터**: 증가 21건 / 감소 4건 — 명암 대비 명확해서 "전월 대비 증감 TOP 3" 시연 시 흥미로움
- **메일 명단**: 정확히 10명 — 슬라이드 프롬프트 요구사항과 1:1 일치
- **팀원 업무**: 진행중 11 / 완료 5 / 지연 2 / 예정 2 — 4가지 상태 모두 있어 대시보드 시각화 풍부

## 결과 재생성

데이터 변경 후 reference outputs를 갱신하려면:

```bash
python3 scripts/generate_reference_outputs.py
```

(스크립트가 없으면 본 README에 명시된 슬라이드 프롬프트를 따라 수동 실행)
